Research
Distilling Motion-Planner Augmented Policies into Visual Control Policies for Robot Manipulation
I-Chun Arthur Liu*, Shagun Uppal*, Youngwoon Lee, Gaurav S. Sukhatme, Joseph J. Lim, Peter Englert
Conference on Robot Learning (CoRL), 2021
Paper / Project / Code

We propose a framework to learn a visual control policy in a two-step procedure: 1) Learn a policy with long-horizon planning capabilities in state space using motion planning-augmented reinforcement learning; 2) Distill this policy into the visual domain while removing the dependency on motion planner.
